# 🐱 にゃんこ大決戦 — Nyanko Wars

にゃんこ大戦争ライクな、横スクロール・レーンバトルゲーム。
HTML5 Canvas + Vanilla JS 製。アセットは AI 画像生成（Pollinations.ai / Flux）で作成。

## 遊び方
- お金が自動でたまる → ユニットボタンで にゃんこを出撃
- 自軍は右へ進軍し、敵と敵拠点を攻撃
- 敵拠点のHPを0にすれば勝利／自拠点が落ちれば敗北
- **財布Lv** を上げると所持金上限と回復速度UP
- **キャットフード爆弾** で画面の敵に大ダメージ（各ステージ3個）
- キーボード: `1`〜`5` で出撃、`F` でフード

### ユニット
| ねこ | 役割 |
|------|------|
| ねこ | 安い量産アタッカー |
| かべねこ | 高HPの壁・前線維持 |
| バトル | バランス型アタッカー |
| アーチャー | 遠距離射撃 |
| 巨神ねこ | 超高コストの範囲フィニッシャー |

### 敵
わんこ / へび / ブタ将軍（重装甲）/ とり（高速）/ 魔王ドグ（ボス・範囲攻撃）

### ステージ
1. はじまりの草原 2. 灼熱キャニオン 3. 極夜の氷原（ボス戦）

## ローカル実行
```bash
npm install        # jimp (アセット生成にのみ必要)
npm start          # http://localhost:5173
```

## アセット再生成（任意）
```bash
npm run gen        # Pollinations.ai で全アセットを再生成 (キー・課金不要)
```

## デプロイ
`public/` を静的ホスティングに置くだけ。
- Netlify Drop: https://app.netlify.com/drop に `public` フォルダをドラッグ
- もしくは `netlify deploy --dir public --prod`

## 技術
- ゲーム本体: `public/game.js`（依存なし）
- アセット生成: `generate-assets.mjs`（Pollinations.ai + jimp 背景除去）
- ローカルサーバ: `server.mjs`
